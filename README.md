# 🧱 LEGO School: 전 세계 레고 팬들을 위한 프리미엄 탐색기

**LEGO School**에 오신 것을 환영합니다! 이 프로젝트는 전 세계 수십만 개의 레고 세트 데이터를 한눈에 살펴보고, 레고에 관한 흥미로운 상식을 배우며 자신만의 즐겨찾기를 관리할 수 있는 고품질 웹 애플리케이션입니다.

> **접속하기:** [https://jvibeschool.org/LEGO/](https://jvibeschool.org/LEGO/)

---

## ✨ 주요 특징 (Key Features)

### 1. 🔍 압도적인 데이터 탐색 (Explore & Themes)
*   **Rebrickable API 연동:** 실시간으로 업데이트되는 방대한 레고 세트 데이터를 탐색할 수 있습니다.
*   **완벽한 시리즈 탐색:** 149개 메인 시리즈와 그에 속한 모든 하위 테마를 빠짐없이 탐색할 수 있습니다.
*   **스마트 정렬 & 검색:** 이름순(ABC) 또는 인기순(Count)으로 시리즈를 즉시 정렬하고, 실시간 검색으로 원하는 시리즈를 빠르게 찾을 수 있습니다.

### 2. 🎢 향상된 사용자 경험 (Enhanced UX)
*   **직관적인 부품수 검색:** 스마트 슬라이더 UI를 통해 부품 수 범위를 정확하게 조절하며, 라벨 겹침 방지 기능으로 시인성을 극대화했습니다.
*   **편리한 네비게이션:** 최적화된 메뉴 순서와 타임머신 연도 표시 확대 등 사용성을 고려한 디테일한 개선이 적용되었습니다.

### 3. ⭐ 레고 상식 100가지 (Trivia Marquee)
*   상단에 흐르는 마키(Marquee)를 통해 레고의 역사, 생산 통계, 미니피규어의 비밀 등 흥미진진한 **100가지 상식**을 만나보세요.
*   클릭 시 상세 모달을 통해 5~10줄 내외의 풍부한 비하인드 스토리를 읽을 수 있습니다.

### 4. 🕒 레고 타임머신 (Time Machine)
*   1950년부터 현재까지, 연도별 슬라이더를 통해 레고의 변천사를 시간 여행하듯 살펴볼 수 있습니다. 특정 연도의 인기 제품을 찾는 재미를 느껴보세요!

### 5. 🧩 부품수 검색 (Part Picker)
*   내가 원하는 크기의 레고를 찾고 싶나요? 부품 수 범위를 조절하여 아주 작은 폴리백 제품부터 수천 개의 조각으로 구성된 매머드급 세트까지 정확하게 검색할 수 있습니다.

### 6. 💖 당신만의 명예의 전당 (Hall of Fame)
*   마음에 드는 세트를 발견했다면 하트 아이콘을 눌러보세요. 나만의 즐겨찾기 목록을 만들어 언제든 다시 찾아볼 수 있습니다.

### 7. 🌍 글로벌 & 멀티 테마 지원
*   **다국어 지원:** 한국어와 영어를 완벽하게 지원합니다.
*   **3가지 테마 모드:** 
    *   **Dark:** 몰입도 높은 세련된 디자인
    *   **Light:** 화사하고 깔끔한 가독성 (가독성 최적화 완료!)
    *   **Gray:** 눈이 편안한 차분한 톤

### 8. 🔗 강력한 외부 링크 연동
*   공식 이미지 조회는 물론, **BrickLink, Brickset, Rebrickable, Google, YouTube**와 한국 최고의 커뮤니티인 **브릭인사이드(Brickinside)**까지 즉시 연결되어 상세 정보를 확인할 수 있습니다.

## 🛠 기술 스택 (Tech Stack)

*   **Frontend:** HTML5, Vanilla JavaScript, CSS3
*   **Design:** Custom Design System (Glassmorphism, Micro-animations)
*   **API:** Rebrickable API
*   **Hosting:** Bitnami Apache Server (jvibeschool.org)
*   **Icons:** Font Awesome, Custom SVG (Piece, Dice)
*   **Fonts:** Inter, Outfit (Google Fonts)
